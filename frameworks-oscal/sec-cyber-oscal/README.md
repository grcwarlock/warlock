# SEC Cybersecurity Disclosure Rules OSCAL Package

OSCAL catalog and profile for SEC Cybersecurity Risk Management, Strategy,
Governance, and Incident Disclosure rules (Final Rule 33-11216).

## Contents

- `catalog/catalog.json` - OSCAL catalog with 20 controls across 3 sections
- `profile.json` - OSCAL profile importing the full catalog

## Sections

| Section | Controls |
|---|---|
| Item 1.05 - Material Cybersecurity Incidents | 5 |
| Item 106 - Risk Management, Strategy, Governance | 12 |
| Annual Reporting (10-K) | 3 |
